# BerylDemon.github.io

ĐỀ BÀI KIỂM TRA THỰC HÀNH HTML & CSS
**Thời lượng:** 4 tiết (mỗi tiết 50 phút)
**Chủ đề:** Xây dựng Website 'Khoa CNTT – Đại học Đại Nam'
1. Mục tiêu
- Vận dụng cấu trúc khối HTML5: header, nav, main, section/article, aside, footer.
- Sử dụng id và class hợp lý để quản lý style.
- Áp dụng các thuộc tính CSS cơ bản (màu sắc, font, kích thước, margin, padding, border, background,...).
- Tạo menu đa cấp (dropdown 2 cấp) bằng HTML & CSS.
2. Yêu cầu cụ thể
1. **Tạo 4 trang web:** index.html, gioi-thieu.html, tin-tuc.html, lien-he.html.
2. Các trang có header, menu, nội dung chính (main), footer.
3. Menu đa cấp gồm các mục: Trang chủ – Giới thiệu – Đào tạo (ĐH, Thạc sĩ, Ngắn hạn) – Sinh viên (CLB, Học bổng, Việc làm) – Tin tức – Liên hệ.
4. Trang Giới thiệu có đoạn văn, danh sách, bảng.
5. Trang Tin tức có danh sách bài viết dạng thẻ (card) có ảnh minh họa.
6. Trang Liên hệ có form nhập Họ tên, Email, Chủ đề, Nội dung và nút Gửi.
7. Dùng CSS để tạo bố cục, màu chủ đạo, font chữ, khoảng cách hợp lý.
8. Menu con hiển thị khi hover, không dùng JavaScript.
3. Cấu trúc thư mục
khoacntt/
├─ index.html
├─ gioi-thieu.html
├─ tin-tuc.html
├─ lien-he.html
├─ css/styles.css
└─ assets/images/
4. Tiến độ thực hiện (4 tiết)
• Tiết 1: Tạo khung trang HTML, liên kết các trang, tạo menu.
• Tiết 2: Tạo menu đa cấp bằng CSS, chỉnh màu, font, khoảng cách.
• Tiết 3: Hoàn thiện nội dung các trang, thêm bảng, form, ảnh.
• Tiết 4: Hoàn thiện bố cục, kiểm tra, nén thư mục nộp bài.
5. Tiêu chí chấm điểm (10 điểm)
1. Cấu trúc & Semantic HTML: 2đ
2. Menu đa cấp (CSS-only): 2đ
3. Bố cục & Trình bày (Flex/Grid): 2đ
4. Tổ chức CSS, id/class: 1,5đ
5. Nội dung trang: 1,5đ
6. Responsive cơ bản: 0.5đ
7. Hoàn thiện, trình bày: 0.5đ
